##Stack
>构造方法

	Stack();
	// 创建一个空堆栈

>特有方法

	E peek();
	// 查看堆栈顶部的对象。
	E pop();
	// 移除堆栈顶部的对象，并为此函数的值返回该对象。
	E push(E item);
	// 把顶压入堆栈顶部。

##LinkedList
>特有方法


	OfferFirst(e); // 插入第一个
	pollFirst(); // 移除最后一个
	peekFirst(); //查看第一个
	
	offerLast(e); // 在末尾插入一个
	pollLast(); // 移除最后一个
	peekLast(); // 查看最后一个
